# SPRING_REACT_MYSQL (Ongoing)

## 📌 프로젝트 소개
SPRING BOOT, REACT, MySQL을 활용한 게시판 프로젝트입니다 (clone coding). 

## 🛠 기술 스택
- **Frontend**: React

## 🚀 실행 방법
```bash
# 프로젝트 클론
git clone https://github.com/hardjava/SPRING_REACT_MYSQL_FRONTEND.git
cd SPRING_REACT_MYSQL_FRONTEND

# 패키지 설치 및 실행
npm install
npm run start
```
